shiso

cute web playable isometric game 

**what??**
----------

you’re trapped in space with no escape? 

you have two options:

*   ponder your existence. with no escape, you have infinite time. what should you do? why is life? what brought you to open this itch link? why even both reading the description?
	
*   if not thought evoking, then this game needs to be fun-evoking. the rule of game dev stipulate that there be a goal. the unwritten lore demands this of you: get rich! lots of gems, and in this economy, you need that money :P go get ur bag twin
	

**have fun exploring!!**

**how to play**
---------------

**avoid the evil red orbs** (definitely not inspired by the pesky little stardew valley insects) (the crossy road ripoff effect was 100% intentional fsfs ofc)

**collect hearts/gems** 

*   hearts → regain health
	
*   gems → moneyyy yayayay
	

**WASD/arrow keys to move around**

**click on inventory slot to use item**

**have fun!**
